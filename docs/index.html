🎯 How It Works

  1. Setup: Place N pins evenly around a circle
  2. Greedy Selection: At each step, try all valid next pins and select the one that creates the line with maximum darkness match
  3. Draw: Draw that line (lighten pixels along the path)
  4. Repeat: Continue until max lines reached or no improvement possible
  5. Export: Generate SVG with precise measurements for physical construction
