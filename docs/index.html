Lessons Learned

  • More pins ≠ better quality: 500 pins optimal, 600 too noisy
  • Edge weighting crucial: 5.0 multiplier gives clear facial features
  • Greedy algorithm ceiling: ~8/10 quality limit without global optimization
  • Simulated Annealing too slow: 5+ minutes vs 8 seconds for greedy
  • Visual validation essential: AI scoring ≠ human perception
