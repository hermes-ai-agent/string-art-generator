🎯 How It Works (v2.0.0)

  1. Preprocessing: Detect edges using Sobel gradient to identify important features
  2. Setup: Place N pins evenly around a circle
  3. Feature-Aware Selection: At each step, try all valid next pins and select the one that best matches BOTH darkness AND edge strength
  4. Draw: Draw that line (lighten pixels along the path)
  5. Repeat: Continue until max lines reached or no improvement possible
  6. Export: Generate SVG with precise measurements for physical construction

🚀 New in v3.2.0: Fixed edge weighting algorithm after systematic debugging:
• Ported Python v2.0.0 scoring logic - score based on remaining darkness, not incremental improvement
• Edge weight 5.0 provides optimal feature prioritization (tested 2.0, 5.0, 7.0)
• Inverted display (white-on-black) improves visibility from 6.5/10 to 7/10
